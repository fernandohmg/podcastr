## Commands

All commands are run from the root of the project, from a terminal:

| Command               | Action                                      |
| :-------------------- | :------------------------------------------ |
| `yarn install`        | Installs dependencies                       |
| `yarn run server-dev` | Starts local dev server at `localhost:3000` |
